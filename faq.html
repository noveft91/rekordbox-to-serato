Frequently Asked Questions

Everything you need to know about R2S Converter, Rekordbox to Serato migration, and how to get started.

What is R2S Converter? ▾

R2S Converter is a native macOS application that converts your entire Rekordbox DJ library to Serato format in seconds. It transfers hot cues with colors, playlists as crates, BPM, musical keys, and all metadata. Works directly from Pioneer USB drives — no Rekordbox installation needed.

How does R2S read my Rekordbox library? ▾

R2S reads Rekordbox data from Pioneer USB drives or exported library folders. You export your library from Rekordbox to USB, then R2S accesses the native Rekordbox database files directly, extracting all metadata, cues, and playlist structure.

What data gets transferred? ▾

R2S transfers hot cues with colors (all 8 positions), playlists and folder structure (as Serato crates and subcrates), BPM values, musical keys (both Camelot and standard notation), metadata (artist, title, album, genre, comments), and star ratings. Memory cues and custom beat grids do not transfer.

Do hot cue colors transfer correctly? ▾

Yes, perfectly. R2S maps all 8 Rekordbox hot cue colors to Serato's 8 color system. Your red cues stay red, blue cues stay blue, and so on. Colors are preserved exactly as you set them in Rekordbox.

Will my playlist folder structure be preserved? ▾

Absolutely. Nested folders in Rekordbox become nested subcrates in Serato. Your entire playlist hierarchy is preserved, so you find tracks the same way you did in Rekordbox.

How long does the conversion take? ▾

R2S processes most libraries instantly to seconds. A 5,000-track library converts in under 2 minutes. A 10,000-track library takes approximately 3-4 minutes. The conversion runs in the background, so you can continue working while it processes.

Does it work with any Pioneer USB? ▾

R2S works with any Pioneer USB drive or Pioneer hardware that supports Rekordbox library exports (Pioneer DDJ, DJM, CDJ controllers, and USB Rekordbox exports). It reads the standard Rekordbox database format, so compatibility is universal across Pioneer products.

What macOS versions are supported? ▾

R2S Converter requires macOS 11.0 (Big Sur) or later. It runs natively on both Intel and Apple Silicon Macs. Older macOS versions may have compatibility issues, so we recommend updating to the latest stable version.

Do I need Rekordbox installed? ▾

No. R2S reads directly from Pioneer USB drives and Rekordbox export folders. Rekordbox does not need to be installed or running. This makes R2S faster and more convenient than other converters.
